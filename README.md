## Пример использования
После запуска приложение:
- Подключается к Elasticsearch
- Создает индекс 'Products'
- Выполняет поиск по префиксам
- Показывает покрытие 96.7%


Инструкция по запуску:
1. git clone https://github.com/DyxDeda/prefix-search-assignment.git
2. cd prefix-search-assignment
3. docker-compose up
